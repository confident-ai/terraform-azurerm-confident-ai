# Security Policy

We take the security of Confident AI and its infrastructure modules seriously.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security problems.

Instead, report privately through one of:

- **GitHub:** the repository's **Security** tab, "Report a vulnerability" (private advisory).
- **Email:** dev@confident-ai.com

Include as much detail as you can: the affected module and version, a description of the issue, steps to reproduce, and any potential impact.

## What to expect

- We aim to acknowledge your report within 3 business days.
- We'll keep you updated while we investigate and prepare a fix.
- Once a fix ships, we're happy to credit you, unless you'd rather stay anonymous.

## Supported versions

Security fixes are provided for the latest released version of this module. Please confirm you're on the most recent release before reporting.

## Scope

This repository contains **Terraform** that provisions cloud infrastructure. It does not include the Confident AI application itself. Issues in the application or the Helm chart should be reported through the main Confident AI project.
